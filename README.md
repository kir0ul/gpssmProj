# Updating
This project is to induce interdependencies among the output values of GPSSMs' transition functions. 

* Implementation of the [Output-Dependent GPSSMs](https://arxiv.org/abs/2212.07608) <br>
* It also includes a benchmark GPSSM reimplementation, namely [PRSSM](https://arxiv.org/abs/1801.10395) <br>
